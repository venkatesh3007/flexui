CachePolicy

Enum defining cache behavior policies.

public enum CachePolicy

Values

ValueDescription
CACHE_FIRSTUse cache if available, fetch in background to update
NETWORK_FIRSTAlways fetch from network first, fallback to cache on failure
CACHE_ONLYOnly use cache, never fetch from network
NETWORK_ONLYAlways fetch from network, never use cache

FlexComponentFactory

Interface for creating custom FlexUI components.

public interface FlexComponentFactory

Methods

create(Context, FlexProps, FlexTheme)

View create(Context context, FlexProps props, FlexTheme theme)

Create a View for this component type.

ParameterTypeDescription
contextContextAndroid context
propsFlexPropsComponent properties from JSON
themeFlexThemeCurrent theme configuration
ReturnsDescription
ViewView instance for this component

Example Implementation

public class CustomButtonFactory implements FlexComponentFactory {
    @Override
    public View create(Context context, FlexProps props, FlexTheme theme) {
        Button button = new Button(context);
        
        // Configure from props
        String text = props.getString("text", "Button");
        button.setText(text);
        
        String color = props.getString("textColor", theme.getColor("text"));
        if (color != null) {
            button.setTextColor(Color.parseColor(color));
        }
        
        return button;
    }
}

Version Information

PackageVersion
dev.flexui:flexui-android1.0.0

Minimum Requirements

  • Android SDK 21+ (Android 5.0)
  • Java 8+ / Kotlin 1.5+
  • Gradle 7.0+

Dependencies

FlexUI has minimal dependencies:

  • Android Support Library (for backward compatibility)
  • OkHttp (for network requests)
  • JSON parsing (built-in Android JSON)

ProGuard/R8 Rules

Add these rules to your proguard-rules.pro file:

# FlexUI
-keep class dev.flexui.** { *; }
-keep class dev.flexui.schema.** { *; }
-keep class dev.flexui.FlexError { *; }

# Prevent obfuscation of component factories
-keep class * implements dev.flexui.FlexComponentFactory { *; }

# Prevent obfuscation of action handlers
-keep class * implements dev.flexui.FlexActionHandler { *; }
