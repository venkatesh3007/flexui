Components

FlexUI comes with 14 essential components that map directly to native Android views. Each component is lightweight, performant, and fully customizable through JSON properties.

Component Reference

  • Container → FrameLayout
  • Row → LinearLayout (Horizontal)
  • Column → LinearLayout (Vertical)
  • Text → TextView
  • Image → ImageView
  • Button → AppCompatButton
  • Scroll → ScrollView
  • List → RecyclerView
  • Grid → RecyclerView + GridLayoutManager
  • Card → CardView
  • Input → EditText
  • Toggle → SwitchCompat
  • Divider → View
  • Spacer → Space

Container

JSON Type "container"
→
Android View FrameLayout

A flexible container that can hold a single child and position it with margins, padding, and alignment.

{
  "type": "container",
  "props": {
    "backgroundColor": "#F5F5F5",
    "padding": 16,
    "marginBottom": 8,
    "cornerRadius": 8,
    "elevation": 2,
    "child": {
      "type": "text",
      "props": {
        "text": "I'm inside a container",
        "textAlign": "center"
      }
    }
  }
}

Properties

PropertyTypeDescription
childObjectSingle child component
backgroundColorStringBackground color (hex or theme variable)
paddingInt/StringPadding for all sides (dp or theme variable)
marginInt/StringMargin for all sides (dp or theme variable)
cornerRadiusInt/StringCorner radius (dp or theme variable)
elevationIntShadow elevation (dp)
widthString"match_parent", "wrap_content", or dp value
heightString"match_parent", "wrap_content", or dp value

Row

JSON Type "row"
→
Android View LinearLayout (horizontal)

Arranges children horizontally in a row with optional spacing, alignment, and distribution.

{
  "type": "row",
  "props": {
    "padding": 16,
    "spacing": 8,
    "mainAxisAlignment": "spaceBetween",
    "crossAxisAlignment": "center",
    "children": [
      {
        "type": "text",
        "props": { "text": "Left", "flex": 1 }
      },
      {
        "type": "text",
        "props": { "text": "Center" }
      },
      {
        "type": "text",
        "props": { "text": "Right" }
      }
    ]
  }
}

Properties

PropertyTypeDescription
childrenArrayChild components
spacingInt/StringSpace between children (dp or theme variable)
mainAxisAlignmentString"start", "center", "end", "spaceBetween", "spaceAround", "spaceEvenly"
crossAxisAlignmentString"start", "center", "end", "stretch"
wrapBooleanWhether to wrap children to next line

Column

JSON Type "column"
→
Android View LinearLayout (vertical)

Arranges children vertically in a column with optional spacing, alignment, and distribution.

{
  "type": "column",
  "props": {
    "padding": 16,
    "spacing": 12,
    "crossAxisAlignment": "stretch",
    "children": [
      {
        "type": "text",
        "props": {
          "text": "Header",
          "textSize": 20,
          "fontWeight": "bold"
        }
      },
      {
        "type": "text",
        "props": {
          "text": "Body content goes here.",
          "textSize": 14
        }
      },
      {
        "type": "button",
        "props": {
          "text": "Action",
          "marginTop": 8
        }
      }
    ]
  }
}

Properties

Same as Row, but arranges children vertically instead of horizontally.

Text

JSON Type "text"
→
Android View TextView

Displays text with full typography control, including fonts, colors, sizes, and styling.

{
  "type": "text",
  "props": {
    "text": "Welcome to FlexUI!",
    "textSize": 18,
    "textColor": "{{colors.primary}}",
    "fontWeight": "bold",
    "textAlign": "center",
    "maxLines": 2,
    "ellipsize": "end",
    "lineSpacing": 1.2
  }
}

Properties

PropertyTypeDescription
textStringText content (supports theme variables)
textSizeInt/StringText size (sp or theme variable)
textColorStringText color (hex or theme variable)
fontWeightString"normal", "bold"
fontStyleString"normal", "italic"
textAlignString"left", "center", "right"
maxLinesIntMaximum number of lines
ellipsizeString"start", "middle", "end"
lineSpacingFloatLine spacing multiplier
letterSpacingFloatLetter spacing (em)

Image

JSON Type "image"
→
Android View ImageView

Displays images from URLs or local resources with scaling, placeholders, and error handling.

{
  "type": "image",
  "props": {
    "src": "https://example.com/image.jpg",
    "width": 200,
    "height": 150,
    "scaleType": "centerCrop",
    "cornerRadius": 8,
    "placeholder": "{{colors.background}}",
    "errorImage": "ic_error"
  }
}

Properties

PropertyTypeDescription
srcStringImage URL or local resource name
widthInt/StringImage width (dp)
heightInt/StringImage height (dp)
scaleTypeString"center", "centerCrop", "centerInside", "fitXY", "fitStart", "fitCenter", "fitEnd"
cornerRadiusInt/StringCorner radius (dp or theme variable)
placeholderStringPlaceholder color while loading
errorImageStringError image resource name

Button

JSON Type "button"
→
Android View AppCompatButton

Interactive button with customizable appearance, text, and actions.

{
  "type": "button",
  "props": {
    "text": "Get Started",
    "backgroundColor": "{{colors.primary}}",
    "textColor": "#FFFFFF",
    "textSize": 16,
    "fontWeight": "bold",
    "padding": 16,
    "cornerRadius": 8,
    "elevation": 3,
    "enabled": true,
    "action": {
      "type": "navigate",
      "url": "/onboarding"
    }
  }
}

Properties

PropertyTypeDescription
textStringButton text
backgroundColorStringBackground color (hex or theme variable)
textColorStringText color (hex or theme variable)
borderColorStringBorder color
borderWidthIntBorder width (dp)
enabledBooleanWhether button is enabled
actionObjectAction to perform on click

Scroll

JSON Type "scroll"
→
Android View ScrollView

Scrollable container for content that may exceed screen bounds.

{
  "type": "scroll",
  "props": {
    "direction": "vertical",
    "showScrollbar": true,
    "child": {
      "type": "column",
      "props": {
        "children": [
          // Long list of items that need scrolling
        ]
      }
    }
  }
}

Properties

PropertyTypeDescription
childObjectSingle child component
directionString"vertical" or "horizontal"
showScrollbarBooleanWhether to show scrollbars
scrollEnabledBooleanWhether scrolling is enabled

List

JSON Type "list"
→
Android View RecyclerView

Efficient scrolling list that recycles views for performance with large datasets.

{
  "type": "list",
  "props": {
    "data": ["Item 1", "Item 2", "Item 3"],
    "itemTemplate": {
      "type": "container",
      "props": {
        "padding": 16,
        "child": {
          "type": "text",
          "props": {
            "text": "{{item}}",
            "textSize": 16
          }
        }
      }
    },
    "spacing": 8,
    "direction": "vertical"
  }
}

Properties

PropertyTypeDescription
dataArrayArray of data items
itemTemplateObjectTemplate for each item
spacingInt/StringSpace between items
directionString"vertical" or "horizontal"
scrollEnabledBooleanWhether scrolling is enabled

Grid

JSON Type "grid"
→
Android View RecyclerView + GridLayoutManager

Grid layout for arranging items in rows and columns.

{
  "type": "grid",
  "props": {
    "data": [1, 2, 3, 4, 5, 6],
    "columns": 2,
    "spacing": 8,
    "itemTemplate": {
      "type": "card",
      "props": {
        "padding": 16,
        "child": {
          "type": "text",
          "props": {
            "text": "Item {{item}}",
            "textAlign": "center"
          }
        }
      }
    }
  }
}

Properties

PropertyTypeDescription
dataArrayArray of data items
itemTemplateObjectTemplate for each item
columnsIntNumber of columns
spacingInt/StringSpace between items
aspectRatioFloatItem aspect ratio (width/height)

Card

JSON Type "card"
→
Android View CardView

Material Design card with elevation, rounded corners, and shadow.

{
  "type": "card",
  "props": {
    "elevation": 4,
    "cornerRadius": 12,
    "backgroundColor": "#FFFFFF",
    "padding": 16,
    "margin": 8,
    "child": {
      "type": "column",
      "props": {
        "children": [
          {
            "type": "text",
            "props": {
              "text": "Card Title",
              "textSize": 18,
              "fontWeight": "bold"
            }
          },
          {
            "type": "text",
            "props": {
              "text": "Card content goes here.",
              "marginTop": 8
            }
          }
        ]
      }
    }
  }
}

Properties

PropertyTypeDescription
childObjectSingle child component
elevationIntShadow elevation (dp)
cornerRadiusInt/StringCorner radius (dp or theme variable)
backgroundColorStringBackground color
strokeColorStringStroke color
strokeWidthIntStroke width (dp)

Input

JSON Type "input"
→
Android View EditText

Text input field with validation, formatting, and various input types.

{
  "type": "input",
  "props": {
    "placeholder": "Enter your email",
    "inputType": "email",
    "required": true,
    "maxLength": 100,
    "backgroundColor": "#F8F9FA",
    "textColor": "#333333",
    "padding": 12,
    "cornerRadius": 6,
    "borderColor": "#E0E0E0",
    "borderWidth": 1,
    "action": {
      "type": "callback",
      "event": "text_changed",
      "data": { "field": "email" }
    }
  }
}

Properties

PropertyTypeDescription
placeholderStringPlaceholder text
valueStringInitial text value
inputTypeString"text", "email", "password", "number", "phone", "url"
maxLengthIntMaximum character length
minLengthIntMinimum character length
requiredBooleanWhether field is required
multilineBooleanWhether to allow multiple lines
maxLinesIntMaximum number of lines (multiline mode)

Toggle

JSON Type "toggle"
→
Android View SwitchCompat

On/off switch for boolean settings and preferences.

{
  "type": "toggle",
  "props": {
    "value": false,
    "enabled": true,
    "trackColor": "{{colors.primary}}",
    "thumbColor": "#FFFFFF",
    "action": {
      "type": "callback",
      "event": "toggle_changed",
      "data": { "setting": "notifications" }
    }
  }
}

Properties

PropertyTypeDescription
valueBooleanInitial toggle state
enabledBooleanWhether toggle is enabled
trackColorStringTrack color when on
thumbColorStringThumb color
actionObjectAction to perform on state change

Divider

JSON Type "divider"
→
Android View View

Visual separator line between content sections.

{
  "type": "divider",
  "props": {
    "orientation": "horizontal",
    "thickness": 1,
    "color": "#E0E0E0",
    "margin": 16
  }
}

Properties

PropertyTypeDescription
orientationString"horizontal" or "vertical"
thicknessIntLine thickness (dp)
colorStringDivider color
lengthInt/StringLength (dp) or "match_parent"

Spacer

JSON Type "spacer"
→
Android View Space

Invisible component for adding space between other components.

{
  "type": "spacer",
  "props": {
    "width": 16,
    "height": 24
  }
}

Properties

PropertyTypeDescription
widthInt/StringWidth (dp or theme variable)
heightInt/StringHeight (dp or theme variable)

Common Properties

All components support these layout properties:

PropertyTypeDescription
idStringUnique identifier for the component
marginInt/StringMargin for all sides
marginTopInt/StringTop margin
marginBottomInt/StringBottom margin
marginLeftInt/StringLeft margin
marginRightInt/StringRight margin
paddingInt/StringPadding for all sides
paddingTopInt/StringTop padding
paddingBottomInt/StringBottom padding
paddingLeftInt/StringLeft padding
paddingRightInt/StringRight padding
widthString"match_parent", "wrap_content", or dp value
heightString"match_parent", "wrap_content", or dp value
flexIntFlex weight (for Row/Column children)
visibilityString"visible", "invisible", "gone"
onClickObjectAction to perform when tapped
